# is-workers
Check if the program is running on Cloudflare Workers

# Usage

```js
var isWorkers = require('is-workers');

console.log(isWorkers()); // => Boolean
```
or
```js
import isWorkers from 'is-workers';

console.log(isWorkers()); // => Boolean 
```

# Installation

With [npm](https://npmjs.org) do

```bash
$ npm install is-workers
```
